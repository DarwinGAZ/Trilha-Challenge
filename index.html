#TrilhaChallenge

O #TrilhaChallenge é uma experiência prática onde você vai transformar um layout profissional em um site funcional — do jeitinho que acontece no mercado. Você vai receber um layout exclusivo criado no Figma, junto com orientações e prazos para colocar a mão no código. Seu objetivo? Reproduzir o layout com fidelidade, aplicando boas práticas de HTML, CSS e JavaScript (ou a stack que você estiver aprendendo). É mais que um exercício: é um passo real rumo ao seu portfólio profissional.

O que vou aprender no desafio?

  • Frontend, Backend, Mobile ou Dados — com orientação pra entender qual faz mais sentido pra você.

  • Nada de pular etapas. Vamos trabalhar lógica, sintaxe, comandos básicos, ferramentas e boas práticas.

  • Você vai colocar a mão no código em desafios práticos com HTML, CSS, JS, ou o que fizer sentido na sua trilha.

  • Além de aprender a programar, você vai criar um portfólio pra mostrar que sabe fazer.
